# PupperMitters

https://puppermittens.herokuapp.com/

A dog lovers e-commerce site for dog “mittens” that allows users to browse products, add them to their virtual cart, and "checkout" on a variety of devices using responsive design.
Developed using: Javascript, React, Redux, React-Redux, Google OAuth, Express, Sequelize, Postgres, Node, and Mocha and Chai for testing.
